# Fora Porta — client scaffold

Static Next.js (App Router) recreation of the **Clients** area of Porta for design iteration. Data is hardcoded in `lib/data.ts` (no database, no API routes).

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — the home route redirects to `/clients`.

## Deploy on Vercel

Connect this repository in the Vercel dashboard and deploy with the default Next.js preset; no environment variables are required.
